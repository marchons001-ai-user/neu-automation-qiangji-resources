# 东北大学自动化强基班课程资料整理

这是由学生整理的非官方课程资料仓库，面向东北大学自动化强基班及相关专业的同学，方便按学年、学期和课程查找学习资料。

维护者：[Marchons001-AIuser](https://github.com/marchons001-ai-user)

[查看完整课程目录](课程目录.md)

## 内容

- `1大一/`：大一课程资料
- `2大二/`：大二课程资料
- `3大三/`：大三课程资料
- `研究生课/`：研究生阶段课程资料
- `开源前脱敏说明.md`：公开前的隐私处理说明

仓库中主要包括学习笔记、作业、实验报告、课程设计、代码以及部分课程资料。目录和文件名以原课程组织方式为主。

## 使用方式

可以直接在 GitHub 中按目录浏览和下载单个文件。仓库包含较多 PDF、Word、PowerPoint 等二进制文档，如只需要某一门课程，建议按需下载，不必下载整个仓库。

## 说明

- 本仓库由学生自发整理，与东北大学及相关院系没有官方隶属或授权关系。
- 内容仅供学习、交流与资料检索，可能存在遗漏、过时或错误，请以任课教师和学校发布的信息为准。

- 资料已进行隐私脱敏，但仍建议贡献者在提交前自行检查姓名、学号、联系方式、账号和文档属性。
- 第三方资料的权利仍归原作者或权利人所有，不因收录到本仓库而改变，详见 [LICENSE.md](LICENSE.md)。

## 贡献

欢迎通过 Issue 补充课程说明、报告错误或提出下架请求。提交新资料时，请确认已经完成隐私检查，并尽量提供资料来源和适用学期。
